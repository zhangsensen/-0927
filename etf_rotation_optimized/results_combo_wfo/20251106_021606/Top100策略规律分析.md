## 💡 五大规律与启示

### 1️⃣ **存在明显的最优频率窗口**

```
最优频率 = 21天（绝对主导）
次优区间 = 7~16天（次优集群）
其他频率 = 零散分布
```

- 21天是绝对最优（29个Top100策略 = 29%）
- 7~16天是次优频率带（总计49个策略 = 49%）
- 其他频率总和才22%

**启示**：不需要测试所有30个频率，只需重点关注3个窗口：**7~16天带** 和 **21天**。

---

### 2️⃣ **简单优于复杂（违反直觉！）**

```
2因子平均Sharpe: 0.8117  ✅ 最优
3因子平均Sharpe: 0.7991  
4因子平均Sharpe: 0.7863  ⚠️ 最低（最多组合反而最差！）
5因子平均Sharpe: 0.7942
```

**启示**：
- 不要过度设计
- 2~3因子的精心选择 > 4~5因子的堆砌
- 少即是多（Less is more）

---

### 3️⃣ **超短期和超长期都不理想**

```
1~6天:   ❌ 完全没有出现（性能太差）
7~21天:  ✅ 主导区间（79个策略 = 79%）
22~30天: ⚠️ 只有少数出现（衰退区间）
```

**启示**：
- 避免日内交易（1~2天）- 成本过高、噪音大
- 避免超长期（>25天）- 均值回归递减
- 最优区间就是 7~21 天

---

### 4️⃣ **存在明显的市场周期效应**

```
 7天 = 1周        → Sharpe 0.7393
13天 = 2周-1天    → Sharpe 0.8003  
21天 = 3周        → Sharpe 0.8817  ⭐ 最优

这不是巧合！与市场自然周期相关：
- 周内效应（5天）
- 双周效应（10天）
- 月度效应（21天）
```

**启示**：测试标准市场周期：7, 14, 21, 28天等，而不是随意选择。

---

### 5️⃣ **WFO排名与实际表现的反向关系**

```
WFO选出的Top 100：全部10天频率
实际表现最优的Top 100：以21天为主（29/100 = 29%）

差异原因：
- WFO是"历史最优"（过度拟合）
- 实际运行是"未来最优"（真实表现）
- WFO和实际结果负相关（p=0.010）
```

**启示**：
- 不完全信任单一评估方法
- 应该进行多频率全面评估
- WFO排名靠后的策略可能真实表现更好
