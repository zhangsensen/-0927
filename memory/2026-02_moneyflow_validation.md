# Moneyflow 因子价值验证：可信负结论

> 日期: 2026-02-17
> 类型: 决策记录 / 可信负结论
> 依据: Rule 29 (可信负结论 > 可疑正结论), Rule 28 (同源变换≠新信息)

---

## 一、背景

Phase 2 数据源探索，评估 Tushare `moneyflow` API（按订单大小分档的 ETF 资金流）能否为 v8.0 提供新的 alpha 维度。moneyflow 按大单/超大单/中单/小单分档，直接编码"谁在买卖"。

理论预期（Rule 30）：如果能区分机构买入 vs 散户卖出，应有预测力。

---

## 二、数据状况

| 数据源 | ETF数 | 时间跨度 | 行数 | WFO可用? |
|--------|-------|---------|------|----------|
| **fund_share** | 49 | 2020-01 ~ 2026-02 | 中位1,488 | 完全满足 |
| **margin** | 43 (pool) | 2020-01 ~ 2026-02 | 50,806 | 完全满足 |
| **moneyflow** | 43 | 2025-08 ~ 2026-02 | **120** | **远远不够** |

- Tushare moneyflow API 仅保留最近约 120 个交易日，无法回溯
- WFO 需要 1,290 天（2020-01 ~ 2025-04），数据缺口 >90%
- **无法进行 WFO→VEC→BT 四重验证**，只能做 IC 分析

## 四、正交性检验（核心结论）

**main_net_pct_5d 与 SHARE_CHG_5D 的截面 rank 相关性: rho = -0.577**

含义：
- r² ≈ 0.33，约 33% 的信息重叠
- 负号含义：机构买入(moneyflow) ↔ 份额流出(fund_share) 同时发生
- **两个数据源在度量同一个现象：机构买+散户卖**

信息来源对比：
```
moneyflow:   交易所实时订单 → 按大单/小单分档 → 当日可得
fund_share:  ETF 申赎披露 → 份额变化 → T+1 可得
margin:      融资融券余额 → 杠杆行为 → T+1 可得

三者都在回答同一个问题: "谁在买？谁在卖？"
只是观测角度不同（订单流 vs 申赎 vs 杠杆），不是不同维度
```

---

## 五、结论：放弃 moneyflow，投资者行为维度已饱和

### 判断逻辑

1. **信号存在但不正交**: IC=0.12 有预测力，但与 SHARE_CHG_5D 相关 0.58 → 同维度
2. **现有因子已充分覆盖**: fund_share (3因子) + margin (2因子) = 投资者行为维度的 5 个度量，且有完整 WFO 验证
3. **数据深度不可弥补**: Tushare 仅保留 120 天，即使从今天开始积累也需 5+ 年才够 WFO
4. **投入产出不合理**: 为 ~33% 的边际信息等 5 年，vs 现有因子已四关验证通过

### 决策

- **moneyflow**: 放弃。不值得积累数据
- **fund_share + margin**: 已是投资者行为维度的最优数据源，无需替代
- **下一步方向**: 应寻找真正正交的维度（与 Price/Volume/InvestorFlow 三维度均正交的信息）

## 六、对未来数据探索的指导（推广 Rule 28）

**Rule 31: 不同管道观测同一现象 ≠ 新信息维度**

```
Rule 28: 同源数学变换≠新信息（SHARE_CHG → SHARE_ACCEL）
Rule 31: 不同管道同一现象≠新信息（moneyflow vs fund_share vs margin）

判断方法: 新数据源与现有因子的截面 rank 相关性
  |rho| < 0.3 → 正交，值得投入
  |rho| 0.3~0.5 → 部分正交，需评估边际成本
  |rho| > 0.5 → 同维度，放弃
  moneyflow: |rho| = 0.577 → 放弃
```

### 已关闭的"看似不同实则相同"数据源

| 数据源 | 与现有因子关系 | 正交? | 结论 |
|--------|-------------|-------|------|
| moneyflow (订单分档) | rho=-0.58 vs SHARE_CHG_5D | 否 | 放弃 |
| ETF申赎 (fund_portfolio) | 与 fund_share 本质相同 | 否 | 放弃 |
| 行业轮动指标 | 被 PP120+BREAKOUT 部分捕获 | 部分 | 低优先 |

### 仍值得探索的方向（预判正交）

| 数据源 | 编码信息 | 预判正交度 | 数据障碍 |
|--------|---------|-----------|---------|
| 北向资金 (stock-level) | 外资配置偏好 | 高（不同投资者群体） | Tushare 仅有市场聚合，无个股分解 |
| 期权隐含波动率 | 前瞻预期 | 高（前瞻 vs 后视） | 仅覆盖 2-3 个标的 |
| 汇率 | 宏观跨资产 | 高（跨资产） | A_SHARE_ONLY 模式下截面区分度存疑 |
| IOPV 折溢价 | 套利/情绪 | 高（微观结构） | 需盘中采集，无历史数据 |
