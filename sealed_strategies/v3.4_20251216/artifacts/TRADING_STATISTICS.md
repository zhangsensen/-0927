## 📈 胜率与盈亏比深度解读

### 为什么胜率 ~50% 是健康的？

```
胜率分级:
  过高 (>60%) → 过拟合！实盘衰减大，不可持续
  健康 (45-55%) → 趋势跟随策略典型特征 ✅
  过低 (<40%) → 赌运气，因子无效
```

**本策略**: 50.79% / 49.42%，处于**最佳区间**

**为什么不追求高胜率？**
1. **过拟合风险**: 胜率 >70% 通常意味着在训练期"记住答案"，实盘会崩
2. **趋势跟随特性**: 本策略是"快速止损 + 放飞利润"，不靠预测准确率
3. **盈亏比补偿**: 胜率 50% + 盈亏比 2.11 = 正期望 +0.55

**反例（过拟合策略）**:
- 训练期胜率: 75%
- 实盘胜率: 45%（衰减 30pp！）
- 盈亏比: 1.2
- 实盘期望: 45% × 1.2 - 55% × 1 = -0.01（亏损！）

---

### 盈亏比 2.11 有多优秀？

```
盈亏比 (Profit Factor) = 总盈利 / 总亏损
```

**分级标准**:
- **< 1.0**: 亏损策略（不可用）
- **1.0 - 1.5**: 勉强盈利（实盘易亏）
- **1.5 - 2.0**: 良好（可用）
- **2.0 - 3.0**: **优秀** ✅ ← 本策略在此区间
- **> 3.0**: 疑似过拟合或样本偏差

**本策略**: 2.11 → 每赚 ¥2.11，只亏 ¥1

**为什么 2.11 是甜蜜点？**
1. **足够高**: 即使胜率 50%，也能稳定盈利
2. **不太高**: 如果 >3.0，可能是过拟合（样本偏差）
3. **可持续**: 长期稳定在 2.0 左右，实盘可复制
